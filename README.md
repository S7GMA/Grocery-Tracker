# Corner Grocer Item Tracking System (C++)
C++ application that ingests transaction data, aggregates item frequencies using maps, and persists results to disk. Features a menu driven interface for queries and text based histograms, demonstrating clean software design, data processing, and reliable file I/O workflows.

## Key Features
- File-based data ingestion
- Frequency aggregation with STL maps
- Item lookup and reporting
- Text-based histogram output
- Persistent storage via file I/O

## Tech Stack
- C++
- STL
- File I/O
- CMake

## Run
```bash
cmake .
make
./CornerGrocer
